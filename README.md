## 🌍 About Travel

**Travel** is an all-in-one travel platform where users can:

- 🎟️ Book tickets  
- 🚗 Rent vehicles  
- 🏨 Reserve hotels  
- 📖 Access travel guides  
- 🎒 Choose customized travel packages  

Built with scalability and user experience in mind.
